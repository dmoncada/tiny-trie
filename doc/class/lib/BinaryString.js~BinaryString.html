    public pointer: Number source

    Word pointer for buffer. With every entry into the buffer, the pointer gets incremented by the entry's width. Every six characters may be encoded, so when the pointer exceeds 6, the buffer can be emptied until the pointer is back under 6.

    Public Methods

    public flush() source

    Encode the remaining items in the buffer. Use this when the input stream is finished to ensure that all data has been encoded.

    public getData(): String source

    Get the binary data as base64. This output does not include padding characters. This procedure flushes the buffer.

    Write a value to the binary string. This value should be thought of as an integer representing the binary data to write.
